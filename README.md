# Pomodoro Timer

A web-based timer for the Pomodoro technique with extra productivity features.

Live demo: https://marionsebastian2023-dot.github.io/pomodoro-timer/

## About
I built this to help with focus during work sessions. It's a simple browser-based timer that follows the Pomodoro method.

## Features
- Customizable work/break durations
- Task list sidebar
- Multiple background themes
- Inspirational quotes
- Audio notifications
- Collapsible interface

## How to Use
1. Set your work time (default 25 minutes)
2. Set your break time (default 5 minutes)
3. Click "Add Session Pair"
4. Press Start
5. Add tasks if needed

## Technology
- HTML/CSS/JavaScript
- GitHub Pages for hosting
- Unsplash for background images
